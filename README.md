# nodejs-web-reactjs-local-json

## Description
A reactjs project that loads local json.

Features: component with parameters and css grid.

## Tech stack
- reactjs
- webpack
- nginx

## Docker stack
- bayesimpact/react-base:latest

## To run
`sudo ./install.sh -u`
- [Avaible here](http://localhost)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
